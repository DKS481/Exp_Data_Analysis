# Exploratory Data Analysis

# Project 1

This repository consists of 8 files:
4x Plots (.PNG)
4x R Files (.R)

The four r files correspond to the four plots.

The data used in these files can be found in the UC Machine Learning Repository at:
https://d396qusza40orc.cloudfront.net/exdata%2Fdata%2Fhousehold_power_consumption.zip

To Run R Files:
1. Download the above data into a folder called Data in your working directory
2. Run R Files
3. Plot PNG images will be saved to the working directory

These files are for the Exploratory Data Analysis Project 1
